## Aperçu du cours

Ce cours couvre les fondamentaux de JavaScript à travers 5 séances progressives, allant des bases du langage jusqu'à l'interaction avec le navigateur.

## Séances de cours

### Séance 1 : Introduction à JavaScript

**Objectifs :**
- Comprendre le rôle de JavaScript
- Exécuter du JavaScript dans le navigateur

**Notions abordées :**
- Différence entre JS et Java
- Console du navigateur
- `console.log` et commentaires
- Variables : `let`, `const`
- Types primitifs : `number`, `string`, `boolean`

### Séance 2 : Logique et contrôle du flux

**Objectifs :**
- Savoir faire des décisions et des répétitions

**Notions abordées :**
- Opérateurs (`+`, `-`, `===`, `!==`, `<`, `&&`, `||`)
- Conditions : `if / else`
- Boucles : `while`, `for`

### Séances 3, 4 & 5 : Fonctions, données et DOM

**Objectifs :**
- Décomposer un problème en sous-problèmes
- Réutiliser et structurer le code
- Manipuler des données
- Relier JavaScript à une interface web

**Notions abordées :**
- Fonctions (déclaration, paramètres, retour)
- Tableaux et objets
- Manipulation du DOM
